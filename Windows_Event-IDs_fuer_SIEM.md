# Windows Event IDs für SIEM

## Anmeldeereignisse

Event-ID 4625 = Logon Failure
Besonders Logontype 3 (Remote-Anmeldung) Hierunter zählen auch Anmeldeversuche per RDP (obwohl in MS-Dokumentation mit Logontype 10 angegeben).
